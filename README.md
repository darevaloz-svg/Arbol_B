# Proyecto Árbol B en Python

## Requisitos

Instalar Graphviz:

https://graphviz.org/download/

Instalar dependencias:

pip install -r requirements.txt

---

## Ejecutar proyecto

python main.py

---

## Cargar archivos CSV

Seleccionar opción 4 del menú e ingresar:

datos/prueba1.csv

o

datos/prueba2.csv

---

## Generar gráfica

Seleccionar opción 6.

La imagen se guardará en:

imagenes/arbol.png
