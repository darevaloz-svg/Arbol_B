# Proyecto Árbol B en Python



## Descripción

Este proyecto implementa un Árbol B en Python configurable por grado, permitiendo almacenar y organizar datos de manera eficiente. El sistema incluye operaciones básicas como inserción, búsqueda y eliminación de datos, además de la carga masiva desde archivos CSV y la generación gráfica del árbol utilizando Graphviz.

El objetivo del proyecto es demostrar el funcionamiento de la estructura de datos Árbol B y su comportamiento en el manejo de grandes cantidades de información.

---

# Funcionalidades

- Configuración del grado del Árbol B.
- Inserción de datos.
- Búsqueda de datos.
- Eliminación de datos.
- Carga masiva desde archivos CSV.
- Visualización de datos almacenados.
- Generación gráfica del Árbol B en formato PNG.


---


## Ejecutar proyecto

python main.py

---

## Cargar archivos CSV

Seleccionar opción 4 del menú e ingresar:

o
